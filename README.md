# KrishiLink
 Farmer’s Growth & Connection Platform

KrishiLink is a modern web application connecting farmers, traders, and consumers in one digital space.  
Instead of a traditional e-commerce platform, KrishiLink works as a social agro network where everyone can interact directly.

For now, every user can:  
- Post what they are growing or selling  
- Browse others’ crop posts  
- Show interest to connect and collaborate

##  Features
- User authentication (Email/Password + Google login)  
- Add, edit, delete crops (CRUD)  
- View all crops with search & filter  
- Crop details with interest form  
- Manage received interests (Accept/Reject)  
- Responsive design (Mobile, Tablet, Desktop)  
- Loading spinners & custom 404 page  

---

##  Tech Stack 
- Frontend: React.js, Vite, Tailwind CSS, React Router  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Deployment: Firebase (Client), Vercel (Server)
